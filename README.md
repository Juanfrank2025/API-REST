# API REST de Gestión de Estudiantes 🎓

Este proyecto es una API RESTful básica construida con **Node.js** y **Express.js**. Su propósito es demostrar el flujo de peticiones y respuestas HTTP mediante la gestión de una lista de estudiantes almacenada en memoria (datos quemados/hardcoded), sin la integración de una base de datos externa. 

Esta es una actividad evaluada para el bootcamp/curso, orientada a dominar los fundamentos del desarrollo backend.

---

## 🚀 Tecnologías Utilizadas

* **Entorno de ejecución:** Node.js
* **Framework:** Express.js (manejo de rutas y middleware `express.json()`)

---

## ⚙️ Requisitos Previos

Asegúrate de tener instalado en tu sistema:
* [Node.js](https://nodejs.org/) (versión 14 o superior recomendada).
